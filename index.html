Audience suggestions, planted from the back row.

SceneSeed gives improv teams a show-specific QR code so the audience can shout suggestions into a phone instead of into the void. Window opens when you say. Closes when you say. Performers see them live.

How it works

Three steps from doors-open to closing scene.

  1. Spin up a show

    Create your improv group, then add a show with a title, prompt, and a submission window.

  2. Share the QR code

    Each show gets a memorable code like coral-2042 and a downloadable QR. Stick it on the program. Project it. Print it on a coaster. Whatever.

  3. Run the room

    Suggestions stream into your dashboard. Favorite, hide, search, and flip to a full-screen performer view when you need to read one out loud.

Why server-locked?

Because frontend locks are theater — anyone with a browser console can bypass them. SceneSeed enforces the submission window in Google's datacenters via Firebase security rules. A forged request bounces off Google's clock, not ours. Your audience plays fair whether they want to or not.
